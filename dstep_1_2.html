Check through SSH

Log onto the unit via SSH to do recovery:

  1. ssh root@X.X.X.X password: (default is cisco)
  2. Creat log Generate logs by running log.sh.

    It will generate a file called debug_log.tar.gz in /tmp directory

  3. Send log to Cisco TAC support for further analysis
    4. 
  4. Issue following commands to recovery the desktop:

    • # /etc/init.d/freedesktop stop
    • # /etc/init.d/freedesktop start  
    • # xfce4-session&
    • # pkill chrome

  5. Once the above commands take effects, you could see CE300 desktop
    for short period and chrome will restart again.
