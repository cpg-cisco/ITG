Reimaging

Please reimagine your CE300, and then try again.

  1. Download the necessary files.

    Download smi-usb-sunbird-1.5rb1-delivery.gz from http://cisco.blob.core.windows.net/image/smi-usb-sunbird-1.5rb1-delivery.gz Download smi.sh from https://dl.dropboxusercontent.com/u/84736663/smi.sh

  2. Copy smi.sh to a functional CE300 unit. Put it on Desktop.
  3. Prepare a USB dirve with at least 2Gb storage.

    Format it to FAT32 on windows. Set “Volume lable” as “SMI-USB”

  4. Copy smi-usb-sunbird-1.5rb1-delivery.gz to SMI-USB drive.
  5. Plug the SMI-USB drive into the side USB port on CE300.
  6. Open a terminal window and change to root privilege with password “cisco”.

    　　　　　

  7. In terminal window input follow commands and run createsmiusb script.

    # cd /apps/localconfig/student/Desktop/ # chmod 777 smi.sh # ./smi.sh

  8. After preparation finished, it should show “finished” in terminal.
  9. Double click “USB SmartInstall” icon on Desktop and follow promote window to finish system update.

  10. Instead of step 8, just hold reset button until CE300 temporary power off, then release reset button. Update should finish in 20 minutes.
