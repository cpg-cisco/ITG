ITG-CPG

recovery

Please check the following steps:

  1. ssh root@X.X.X.X

    password: (default is cisco)

  2. Create log Generate logs by running log.sh.

    It will generate a file called debug_log.tar.gz in /tmp directory

  3. Send log to Cisco TAC support for further analysis

    a) # /etc/init.d/freedesktop stop
    b) # /etc/init.d/freedesktop start
    c) # xfce4-session&
    d) # pkill chrome

  4. Once the above commands take effects, user could see CE300 desktop for short period and chrome will restart again.

     

  5. Once chrome is restarted, the user should utilize the latest “AppSpace Edge 300 Wizard” from AppSpace to reinstall AppSpace player.

     

  6. After reinstalling the AppSpace player, the CE300 should be back to work.

     

  • *Note: The latest version of “AppSpace Edge 300 Wizard” is 5.0.3 v3.
