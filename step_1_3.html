Check through telnet

Please check the following steps:

  1. Hold “reset” button until CE300 power off, then release “reset” button.

     

  2. After reboot, CE300 should enter factory mode, the output of CE300 will stay at Stage 1 with only the white Cisco logo being displayed

     

  3. Telnet to access CE300 with the recorded IP address telnet X.X.X.X

    a) Account name: root
    b) Password: (default is cisco)

  4. Create log Generate logs by running log.sh.

    It will generate a file called debug_log.tar.gz in /tmp directory

  5. Send log to Cisco TAC support for further analysis

     

  6. Issue “power off” command

     

  7. Once all the LED panel lights are off, unplug/replug power

     

  8. Press Power button to start the unit
