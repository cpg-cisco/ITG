Check through SSH

Please check the following steps:

  1. Log onto the unit via SSH to do recovery

    SSH:root@X.X.X.X

    password:(default is Cisco)

  2. Create log

    Generate logs by running log.sh. It will generate a file called debug_log.tar.gz in /tmp directory

  3. Send log to Cisco TAC support for further analysis
  4. Issue following commands to recovery the desktop:

    # /etc/init.d/freedesktop stop

    # /etc/init.d/freedesktop start

    # xfce4-session&

    # pkill chrome

  5. Once the above commands take effects, you could see CE300 desktop for short period and chrome will restart again.
  6. Once chrome is restarted, please utilize the latest “AppSpace Edge 300 Wizard” from AppSpace to reinstall AppSpace player.
  7. After reinstalling the AppSpace player, the CE300 should be back to work.

    *Note: The latest version of “AppSpace Edge 300 Wizard” is 5.0.3 v3.
